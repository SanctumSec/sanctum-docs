# Getting Started

Get from zero to a working credential vault connected to your AI editor in under 10 minutes.

## Prerequisites

- macOS, Linux, or WSL2
- An AI editor that supports MCP (Cursor, Claude Code, Windsurf, VS Code)

## Steps

1. **[Installation](/getting-started/installation)** — Install the Sanctum CLI
2. **[Quickstart](/getting-started/quickstart)** — Initialize your vault and store your first secret
3. **[Your First Credential](/getting-started/first-credential)** — Register an agent and create access policies
4. **[Connect to AI Tool](/getting-started/connect-ai-tool)** — Wire up MCP to your editor

Each step builds on the previous one. By the end, you'll have a fully functional credential vault with policy-controlled access for your AI tools.

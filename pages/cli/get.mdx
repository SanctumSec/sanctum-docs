# sanctum get

Retrieve a credential value from the vault.

## Usage

```bash
sanctum get <path>
```

## Arguments

| Argument | Description |
|----------|-------------|
| `path` | Credential path (e.g., `openai/api_key`) |

## Example

```bash
sanctum get openai/api_key
```

```
sk-proj-abc123...
```

The raw value is printed to stdout with no extra formatting, making it suitable for piping:

```bash
export OPENAI_API_KEY=$(sanctum get openai/api_key)
```


> **Note:**
`sanctum get` outputs the raw secret to stdout. For AI agent access, use the MCP server instead — it enforces policies and creates audit entries with agent identity.


## Notes

- The operation is recorded in the audit log as `credential.retrieve`
- Exit code 0 on success, 1 if the credential doesn't exist
